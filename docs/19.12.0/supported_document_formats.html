Supported File Formats

With GroupDocs.Viewer for .NET you can view over 140 different document types and auto-detect over 90 file formats. Auto-detection means that files that are missing an extension from their filenames can be identified by GroupDocs.Viewer. Please refer to the following supported formats list for more information:
