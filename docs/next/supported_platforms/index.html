Supported Platforms

GroupDocs.Viewer for .NET supported platforms are listed below. For more platform support please check GroupDocs.Viewer for Java.

PlatformDescription
.NET Standard 2.0Any type of .NET Standard 2.0 application is supported (since Q4 2019).
.NET FrameworkAny type of .NET Framework version 2.0 (or later) application is supported - ASP.NET, Web Services, WinForms, WPF and others. Full support for 32-bit and 64-bit.
Windows AzureYes, GroupDocs.Viewer for .NET runs on Windows Azure.
MonoYou can use GroupDocs.Viewer for .NET to build applications with Mono.
